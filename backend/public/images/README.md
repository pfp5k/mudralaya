# Images Folder

Place all your images in this folder. The following images are referenced in the application:

## Required Images:

1. **mudralya_logo.webp** - Logo used in Header and Footer
2. **advisor-img.png** - Hero section image
3. **what_to_do.webp** - "What You'll Do" section image
4. **grow_1.webp** - "From Advisor to Leader" section image
5. **grow_2.webp** - "We Empower You" section image
6. **grow_3.webp** - "Who Can Join" section image
7. **Empowering_Women.webp** - Empowering Women section image
8. **aboutus.webp** - About Us page main image
9. **aboutus-1.gif** - Mission card 1 animated icon
10. **aboutus-2.gif** - Mission card 2 animated icon
11. **aboutus-3.gif** - Mission card 3 animated icon
12. **conact-us.webp** - Advisor form background image

## Image Paths:

All images should be placed directly in this `public/images/` folder and will be accessible via:
- `/images/filename.webp`
- `/images/filename.png`
- `/images/filename.gif`

## Notes:

- Images in the `public` folder are served from the root URL
- Use webp format for better performance when possible
- Ensure images are optimized for web use
- Recommended max width: 1920px for hero images, 800px for content images
